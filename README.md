# Dungeon-game
Rouguelike in Python 3.
download python 3 from http:www.python.org

I use this to learn Python


## how to play

use aswd to move 
use "up" and "down" to go down and up from the floor

### items are in lower case: 
    f..... food
    k ..... key
    $ ...... gold
    b ....... box

### Monsters are in upper case:
    M ...... Monster
    B ...... Bat 
    D ...... Dragon
